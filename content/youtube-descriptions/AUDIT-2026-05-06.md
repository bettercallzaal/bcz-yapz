# YouTube Description Audit - 2026-05-06

Pulled all 18 published BCZ YapZ episodes via `yt-dlp` and compared their live YouTube titles + descriptions against the new format (title spec + chapters at bottom + 4 standard blocks).

## TL;DR

**17 of 18 episodes need an upgrade.** Only Episode 18 (Andy Minton, Hangry Animals) already matches the new spec - Zaal hand-rolled it with the new format.

Per-ep upgrade patches are pre-generated at `content/youtube-descriptions/_upgrades-2026-05-06/<slug>.md`. Each patch file has:
- The new title (paste over the existing YouTube title)
- The fully upgraded description body (paste over the existing description)

The patches preserve Zaal's original prose, fix em dashes to hyphens, fix any Warpcast -> Farcaster, and append the 4 standard footer blocks (`MENTIONED IN THIS EPISODE`, `FOLLOW {{guest}}`, `BCZ YAPZ`, `THE ZAO`) above the chapters block, then put chapters at the bottom.

## How to apply

1. Open YouTube Studio, find the episode
2. Open `content/youtube-descriptions/_upgrades-2026-05-06/<slug>.md`
3. Copy the new title -> paste over the YouTube title
4. Copy the upgraded description body -> paste over the YouTube description
5. Save

Order of operations: I'd start with the oldest (ep 1) and work forward, since the older ones are the worst-formatted and benefit the most from the upgrade.
